# Arquitetura - SEXTA FEIRA BOT

Arquitetura base scaffold criada no workspace.

Principais componentes:

- core/ - modelos e processadores
- ai/ - ConversationManager e stubs para vetores/agents
- database/ - stubs para DB e migrations
- interfaces/ - Telegram integrado e terminal (stubs)
- utils/ - logger e helpers
- payments/ - stub para Mercado Pago
- scripts/ - scripts utilitários (run_tests.py)

Use `scripts/run_tests.py` para executar os testes iniciais.
